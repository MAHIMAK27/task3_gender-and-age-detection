# 👤 Gender & Age Detection System

A simple web-based application that detects a person's **gender and age** from an image using deep learning models.

---

## 🚀 Features
- 👁️ Face detection from image
- 🧑 Gender prediction (Male/Female)
- 🎂 Age estimation (age range)
- 🌐 Simple and interactive web interface

---

## 🛠️ Tech Stack
- Python
- OpenCV
- Deep Learning (CNN)
- HTML
